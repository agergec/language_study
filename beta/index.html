Help

GETTING STARTED
Load words: Add one or more .txt files. Each line is a word or phrase.
Ways to add files: Welcome screen → “Add Word List…”, Menu → Data & System → Files → “Add Word List…”, or drag & drop anywhere.
After loading: the app builds your word pool and enables study controls.
STUDY MODES
Listen & Read
Shows text and speaks once.
Read then Say
You read first. On Next, the app speaks once.
Listen then Repeat
App speaks, you repeat. On Next, reveals and speaks again.
Articles — le/la
Shows base (e.g., “… table”). Pick the correct definite article.
Articles — un/une
Shows base (e.g., “… ami”). Pick the correct indefinite article.
Switch mode: Menu → General → Study Mode or tap the mode text in the footer.
Availability: Article modes become selectable when matching words exist.

VOICES & AUDIO
Voice selection: Menu → Voice Configuration → Voice. Tap “Test Voice” to preview.
System voice: Uses your browser/OS voices for French (fr‑FR).
OpenAI / ElevenLabs: Add an API key in Integrations to enable extra voices.
Quick change: Tap the voice text in the footer to pick a voice inline.
Repeat speed‑up: The app caches the most recent TTS clip temporarily to replay instantly on “Repeat”. Cache clears on mode/voice/list/shuffle changes and on app start.
FILES & PERSISTENCE
Manage files: Menu → Data & System → Files. See counts, enable/disable, delete (×), and add.
Enable/disable: Toggles take effect immediately and rebuild the session.
Delete: Use the × button to remove a file (also removed from local storage).
Persistence: On first load per session, you can opt to save lists to IndexedDB. If accepted, lists restore automatically on next visit.
Reset app: Settings → Reset clears local/session storage, IndexedDB word lists, and cached files.
STATUS BAR QUICK ACTIONS
Mode
Tap the text to switch modes inline.
Voice
Tap the text to pick a voice inline.
Files
Tap the icon to see file list and add a new list.
Shuffle
Tap to toggle. A compact confirmation appears near the icon.
INTEGRATIONS
OpenAI: Add a .key file in Menu → Integrations → OpenAI. Remove with the trash icon.
ElevenLabs: Add a .key file in Menu → Integrations → ElevenLabs. Remove with the trash icon.
Notes: Keys are stored locally on your device. Network is required for these voices; if unavailable, the app falls back to the system voice.
TIPS & TROUBLESHOOTING
• If audio does not play, try interacting (tap/click) once to satisfy autoplay policies.
• If voices are missing, open the Menu and revisit Voice to refresh the list.
• If article modes are disabled, ensure your lists include words starting with the required prefixes (le/la or un/une).
• On desktop, collapse/expand the Answers panel with the chevron. Use the History button to see all revealed items.
